Data Layers with Scale Dependencies

Most data layers are visible at all scales, but some data layers have scale dependencies and are only visible within certain scale ranges, such as less than 1:100,000. If a data layer has a scale dependency and the map is not within the appropriate scale range, the data layer will not draw on the map and its icon in the "Active Data Layers" window will have a yellow exclamation point. Hover the mouse over the yellow exclamation point to see the range within which the data layer will draw. To view the data layer, zoom in or out to a map scale within the available scale range.
