User Guide
MORIS, the Massachusetts Ocean Resource Information System, is an online mapping tool created by the Massachusetts Office of Coastal Zone Management (CZM), the Office of Geographic Information (MassGIS), SeaPlan (formerly Massachusetts Ocean Partnership), Applied Science Associates (ASA), Charlton Galvarino, and PeopleGIS. MORIS can be used to search, display, and share spatial data pertaining to Massachusetts coastal and ocean resources. Users can quickly create and share maps and download data for use in a Geographic Information System (GIS). MORIS connects to MassGIS's GeoServer-based web mapping service.

Terms and Conditions: By using MORIS you agree to the following terms.

*Please note: MORIS is one of a family of applications, all built on the same template. Some of the other versions include OLIVER (the MassGIS Online Data Viewer) and MuniMapper. This user guide applies to all of these applications, so you may substitute "OLIVER," "MuniMapper," etc. for "MORIS" as you read this guide.
