Removing Data Layers

To temporarily turn a data layer off, uncheck the box next to the data layer name in the "Active Data Layers" window. The "Uncheck all" button may be clicked to turn off all the active data layers. The legends of unchecked data layers will be removed from the "Legend" window.

To remove a data layer, right-click the data layer name in the "Active Data Layers" window and select "Remove layer." To remove more than one data layer, hold the Ctrl key on your keyboard to highlight multiple rows, right-click, and select "Remove layer." The "Remove all" button may also be clicked to remove all the active data layers.
