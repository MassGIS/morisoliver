User Guide

Adding Data Layers by Searching

  1. Type a search term in the "Search data layers" box in the "Available Data Layers" window. The application searches data layer names and lists all data layers with that term in the title.

  2. Click the data layer name of interest.

  3. The data layer is drawn on the map. The folders in the "Available Data Layers" window will open to the first listing of the data layer. The data layer name will appear in the "Active Data Layers" window and the legend will be added to the "Legend" window.
