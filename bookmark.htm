Bookmarking/Sharing a Map

  1. To create a bookmark to launch the application with the current map settings (e.g., data layers, extent, etc.) enabled, click the Permalink tool.

  2. Right-click the blue permalink and see the options. In Firefox "Copy Link Location" will copy the URL to the clipboard. "Bookmark This Page" will create a Firefox bookmark. In Internet Explorer "Copy Shortcut" will copy the URL to the clipboard. "Add to Favorites" will create an Internet Explorer favorite. This URL may be saved as a bookmark in the web browser or shared with others in an email to launch the current map (includes data layers, extent, basemap, and map units only, but not selected features and opacity settings).
