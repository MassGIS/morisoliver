Searching for a Location using the MassGIS base streets dataset

    The base streets dataset may have fresher street information than the Bing search location service. However, addresses must be entered into a form as free form address format interpretation is not available.
  1. Click the red pin button to open the address form.
  2. Enter two or more pieces of address information.



  3. The search results box will appear, with a score.


    4. If this is the correct results, click the "Zoom to center point" button for the map to zoom to the address.
  4. Click the "Clear location" button to clear the location search results (the blue balloon) from the map.
